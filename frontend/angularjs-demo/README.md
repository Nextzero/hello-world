AngularJS实战

-----------------------------------------------------------------------

目标：
    通过代码实例，学习AngularJS

-----------------------------------------------------------------------
版本：
    AngularJS v1.4.6
    Bootstrap v3.3.7
    jQuery v2.1.4

-----------------------------------------------------------------------    

部署：
    若使用templateUrl，angular会通过网络获取url指向的内容，因此在本地调试时需要配置一个web服务器。
    在例子中，使用的是nginx，根路径配置成了angular-demo这一级目录。
    访问示例如：http://localhost/5.指令/1.入门.html
